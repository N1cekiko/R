# ** 数据类型**   
## **1.数值型**    
赋值:“<-”、“=”   
“==”判断是否相等   
class()该命令用于看数据类型   
```r
class(data4$duration)#查看文件data4的日期这一列的类型。
exp(1000)#正无穷
-10/0 #负无穷
```
